# 老板端打卡微信小程序

## 项目介绍

这是一个微信小程序版本的老板端打卡系统，用于老板拍照记录工人上班打卡情况，管理工人出勤，查看数据分析。

### 功能特点

- 📷 **拍照签到**：拍摄工人照片，自动添加时间、地点水印
- 👥 **工人管理**：添加、编辑、删除工人信息
- 📋 **签到记录**：查看历史签到记录，按日期、工人筛选
- 📊 **数据分析**：查看出勤统计、迟到分析、出勤率趋势
- 🎯 **简洁界面**：大按钮、清晰文字，适合文化水平不高的用户使用

## 权限声明

小程序需要以下权限：

- **相机权限**：用于拍摄工人打卡照片
- **位置权限**：用于记录打卡地点信息

## 技术栈

- 微信小程序原生框架
- 本地存储（wx.setStorageSync/wx.getStorageSync）
- 相机API（wx.createCameraContext）
- 位置API（wx.getLocation）

## 数据存储

- 工人信息存储在本地存储中
- 签到记录存储在本地存储中
- 数据不会上传到服务器，保证数据隐私

## 使用方法

### 1. 导入项目

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择项目目录：`E:\打卡系统\wechat-miniprogram`
4. 填写AppID（如果没有，选择「测试号」）
5. 点击「导入」

### 2. 运行项目

1. 在微信开发者工具中点击「编译」按钮
2. 点击「预览」按钮，使用微信扫码查看
3. 或点击「真机调试」按钮，在手机上查看

### 3. 主要功能使用

#### 拍照签到
1. 点击底部导航栏的「拍照签到」按钮
2. 选择要签到的工人
3. 点击「开始拍照」按钮
4. 拍摄工人照片（自动添加时间、地点水印）
5. 点击「确认签到」按钮完成签到

#### 工人管理
1. 点击底部导航栏的「工人管理」按钮
2. 点击「添加工人」按钮，输入工人信息
3. 点击工人列表中的工人，可以编辑或删除工人

#### 签到记录
1. 点击底部导航栏的「签到记录」按钮
2. 可以按日期、工人筛选记录
3. 点击记录可以查看详情和照片

#### 数据分析
1. 点击底部导航栏的「数据分析」按钮
2. 可以查看周、月、年的出勤统计
3. 可以查看出勤率趋势和迟到分析

## 注意事项

1. **权限说明**：首次使用时会请求相机和位置权限，请务必授权，否则无法正常使用拍照和定位功能

2. **数据安全**：所有数据存储在本地，卸载小程序会导致数据丢失，建议定期备份数据

3. **网络要求**：小程序可以在无网络环境下使用，但位置信息获取需要网络

4. **相机使用**：拍照时请确保光线充足，避免背光，以保证照片清晰

5. **位置信息**：位置信息用于记录打卡地点，不会用于其他用途

## 常见问题

### Q: 拍照时无法启动相机

A: 请检查是否已经授权相机权限，在微信「设置」-「隐私」-「相机」中查看是否允许小程序使用相机

### Q: 无法获取位置信息

A: 请检查是否已经授权位置权限，在微信「设置」-「隐私」-「位置信息」中查看是否允许小程序使用位置信息

### Q: 数据丢失怎么办

A: 由于数据存储在本地，卸载小程序会导致数据丢失。建议定期截图保存重要数据，或使用微信开发者工具导出数据

### Q: 小程序卡顿怎么办

A: 请尝试清理微信缓存，或重启微信后再次使用
