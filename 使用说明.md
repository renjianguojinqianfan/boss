# 老板打卡系统微信小程序使用说明

## 一、小程序介绍

老板打卡系统微信小程序是一款专为建筑工地方老板设计的工人考勤管理工具，通过拍照签到的方式，帮助您轻松管理工人出勤情况，记录工作时间，生成考勤报表，让工地管理更加规范、高效。

### 主要特点：
- 拍照签到：工人签到时必须拍照，确保真实出勤
- 位置记录：自动记录签到地点，防止代签
- 数据管理：方便查看和管理工人信息
- 考勤分析：自动生成考勤报表，掌握工地出勤情况

## 二、如何使用

### 1. 打开小程序

在微信中搜索「老板打卡系统」或扫描下方二维码，点击进入小程序。

### 2. 首次使用设置

首次使用时，系统会请求以下权限，请点击「允许」：
- 相机权限：用于拍摄工人签到照片
- 位置权限：用于记录签到地点

### 3. 添加工人信息

在使用签到功能前，需要先添加工地工人信息：

1. 点击底部导航栏的「工人管理」
2. 点击右上角「+」按钮
3. 填写工人姓名、电话等信息
4. 点击「确定」保存

### 4. 开始签到

1. 点击底部导航栏的「打卡」
2. 从下拉菜单中选择要签到的工人
3. 点击「拍照」按钮，对准工人拍摄照片
4. 确认照片无误后，点击「确认签到」

### 5. 查看签到记录

1. 点击底部导航栏的「记录」
2. 可以查看所有工人的签到记录
3. 点击单条记录可以查看详细信息和照片

### 6. 查看考勤分析

1. 点击底部导航栏的「分析」
2. 系统会自动生成考勤报表
3. 可以查看工人出勤情况和统计数据

## 三、功能说明

### 1. 首页

显示系统欢迎信息和快捷功能入口，方便您快速进入各个功能模块。

### 2. 工人管理

- **添加工人**：点击「+」按钮，填写工人信息并保存
- **编辑工人**：点击工人列表中的「编辑」按钮，修改工人信息
- **删除工人**：点击工人列表中的「删除」按钮，确认后删除工人
- **搜索工人**：在搜索框中输入工人姓名或电话，快速找到工人

### 3. 打卡功能

- **选择工人**：从下拉菜单中选择需要签到的工人
- **拍照签到**：点击拍照按钮，拍摄工人照片，系统会自动添加时间、地点水印
- **确认签到**：检查信息无误后，点击确认完成签到
- **重复签到提醒**：系统会自动检查，防止工人重复签到

### 4. 记录管理

- **查看记录**：显示所有工人的签到记录，按时间倒序排列
- **记录详情**：点击单条记录，查看详细的签到信息和照片
- **筛选记录**：可以按日期或工人筛选记录

### 5. 数据分析

- **出勤统计**：显示工人出勤天数和出勤率
- **部门分析**：按部门统计出勤情况
- **趋势分析**：显示近期出勤趋势图表

## 四、常见问题解答

### 1. 为什么拍照失败？

- 请检查手机相机是否正常工作
- 确保小程序已获得相机权限
- 确保手机存储空间充足
- 尝试重新进入小程序

### 2. 为什么位置信息获取失败？

- 请检查手机定位服务是否开启
- 确保小程序已获得位置权限
- 尝试在开阔地带使用，确保GPS信号良好

### 3. 如何恢复误删除的工人信息？

- 目前系统不支持恢复已删除的工人信息
- 建议在删除前仔细确认
- 可以重新添加被删除的工人信息

### 4. 签到记录可以导出吗？

- 目前系统支持在「分析」页面查看统计数据
- 后续版本将支持导出Excel格式的考勤报表

### 5. 小程序需要联网使用吗？

- 拍照签到功能在无网络情况下也可以使用
- 但建议在有网络时使用，确保数据同步
- 网络恢复后，系统会自动同步离线时的签到数据
