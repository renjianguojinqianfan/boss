## 使用说明

### 1. 首次使用

1. **打开小程序**：扫描二维码或搜索小程序名称
2. **授权权限**：允许相机和位置权限
3. **添加工人**：在"工人管理"页面添加工人信息
4. **开始签到**：在首页点击"拍照签到"

### 2. 签到流程

1. **选择工人**：从下拉列表中选择需要签到的工人
2. **查看位置**：确认当前位置信息正确
3. **拍摄照片**：对准工人点击"拍照"按钮
4. **确认照片**：查看拍摄的照片，确认水印添加正确
5. **完成签到**：点击"确认签到"按钮
6. **查看记录**：自动跳转到签到记录页面查看

### 3. 查看记录

1. **进入记录页面**：在首页点击"签到记录"
2. **浏览记录**：查看所有签到记录，按时间倒序排列
3. **查看详情**：点击记录项查看详细信息

## 常见问题及解决方案

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 相机初始化失败 | 设备相机不可用 | 检查设备相机是否正常 |
| 拍照失败 | 相机权限未授权 | 在系统设置中开启相机权限 |
| 位置获取失败 | 位置权限未授权 | 在系统设置中开启位置权限 |
| 无工人可选 | 未添加任何工人 | 在工人管理页面添加工人 |
| 签到失败 | 数据保存失败 | 检查存储空间是否充足 |
| 水印添加失败 | Canvas操作失败 | 系统会自动使用原图保存 |
